Semáforo

Na aula 5, nós revisitamos o site Tinkercad para criar um semáforo utilizando o Arduino. Foi interessante voltar a usar essa plataforma que já conhecíamos de outras aulas. Com a orientação do professor, montamos os componentes no ambiente virtual do Tinkercad. Em seguida, escrevemos o código para controlar o semáforo usando a linguagem de programação do Arduino.

Após recriarmos o semáforo virtualmente no Tinkercad, avançamos para uma abordagem física utilizando o programa Arduino IDE. Com a orientação do professor, montamos o circuito novamete e colamos o codigo utilizado no tinkercad no Arduino IDE para controlar o semáforo diretamente na placa Arduino.
